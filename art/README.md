# art/ — 画像を入れる場所

このフォルダに、下の**決まったファイル名**で画像をアップロードしてください（GitHubモバイルアプリ＝Add file → Upload files。**同じ名前で上書き**すればコードは触らず反映）。

| ファイル名 | 用途 | サイズ |
|---|---|---|
| `hero.jpg` | 開幕（全画面ヒーロー） | 1080 × 2340 px（9:19.5） |
| `mid1.jpg` | 中間① 上ステージ | 1080 × 1170 px（12:13） |
| `mid2.jpg` | 中間② 上ステージ | 1080 × 1170 px |
| `mid3.jpg` | 中間③ 上ステージ | 1080 × 1170 px |
| `ending.jpg` | 結末（全画面ヒーロー） | 1080 × 2340 px（9:19.5） |

- JPEG・各〜400KB目安（重いと携帯で遅い）。
- 全画面（hero/ending）は下1/3に文字が重なるので主役は上〜中央に。
- 中間（mid）は下辺が文字帯へグラデで溶けるので端に情報を詰めない。
